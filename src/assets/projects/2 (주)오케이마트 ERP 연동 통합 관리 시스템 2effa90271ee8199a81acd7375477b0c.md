## ***Overview***

---

**중소 이커머스 기업의 엑셀 기반 수작업 업무를 자동화 시스템으로 전환하여 멀티채널 판매와 ERP 연동을 지원하는 프로젝트**

국내 온라인 쇼핑 거래액이 연평균 11.9% 증가하며 멀티채널 판매가 필수가 된 상황입니다. 많은 이커머스 기업들은 여전히 Excel 기반 수작업으로 인해 운영 비효율성과 휴먼에러(실수, 실패) 문제가 반복되고 있습니다. 정부의 디지털 뉴딜 정책과 AI 기술 투입이 가속화되고 있으나 중소기업의 DX 전환은 기술 접근성과 비용 문제로 지연되고 있는 실정입니다. 

- 멀티 쇼핑몰 API 활용 통합 쇼핑몰 시스템 구축
    - 기존 Excel 매크로 기반 업무를 Python FastAPI로 1차 개선한 성과를 바탕으로, 멀티 쇼핑몰 API 연동을 통한 통합 주문/재고 관리 시스템 개발
- ERP API 연동 자동화 시스템
    - 디지털 트랜스포메이션(DX)가속화와 AI 기술 발전을 활용한 이커머스 ERP 연동 자동화
- **(주)오케이마트** 사내 배포를 완료하였으며, 현재 사내 직원들 실사용 중

## *Problem Solving*

---

### 1. 엑셀 상품 데이터 추출 컬럼 매핑 이슈

- **문제**: DB에 저장된 컬럼명과 엑셀 파일로 추출해야 하는 컬럼명이 서로 달라 매핑이 어려운 문제 발생
- **해결**:
    - DB 컬럼과 엑셀 컬럼 간 매핑 로직을 분리하여 엑셀 생성 시 자동 매핑하도록 구조 개선
    - 엑셀 추출용 컬럼명을 템플릿 형태로 코드에 사전 정의
    - DB 스키마 변경과 무관하게 엑셀 포맷을 안정적으로 유지 가능하도록 개선
