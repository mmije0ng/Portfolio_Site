## **Overview**

---

**스마트폰 사용 증가로 저하된 키보드 타이핑 능력을 회복하고, 문해력과 어휘력을 동시에 향상시키는 React 기반 웹 서비스**

- 타이핑 속도 및 정확도 향상을 위한 연습 모드 제공
- 문단 내 주요 키워드 설명과 퀴즈를 연계하여 학습 동기 부여 및 지식 습득 유도
- 학습 요소와 게임적 요소를 결합하여 지속적인 사용자 참여 유도

1. **타이핑 결과 분석 및 UI 구현**: 타이핑 결과 페이지의 전반적인 UI를 설계하고, 학습 성과를 한눈에 파악할 수 있는 대시보드 구현
2. **정밀 타이핑 측정 알고리즘 개발**: 실시간 타이핑 속도(WPM), 정확도, 오타 수, 진행도를 정확하게 계산하는 알고리즘 구현
3. **오타 패턴 시각화 로직 구축**: 사용자가 자주 틀리는 키를 데이터 기반으로 분석하고, **가상 키보드(Virtual Keyboard) 내 시각적 강조 표시**를 통해 취약 구간 인지 유도
4. **학습 콘텐츠 연동**: 타이핑 문단과 키워드 퀴즈를 매끄럽게 연결하는 인터렉션 로직 설계

## *Problem Solving*

---

### 1. 한글 타이핑 특성상 자소 단위 입력 처리 및 오타 판별 로직 구현

- **문제**: 한글은 초성·중성·종성이 결합되는 조합형 문자로, 단순 문자열 비교만으로는 실시간 오타 판별과 정확한 진행도 계산에 한계가 있음
- **해결**:
    - **hangul-js 활용**: 입력 데이터를 자소 단위로 분리하여 기준 문단과 실시간으로 비교하는 로직 적용
    - **실시간 피드백 시스템**: 입력 값과 기준값을 자소 단위까지 정밀하게 대조하여 즉각적인 오타 판별 및 진행도 갱신이 가능하도록 개선
    - **UX 개선**: 오타가 발생한 지점을 시각적으로 강조하여 사용자가 자신의 타이핑 습관과 오타 패턴을 쉽게 파악할 수 있도록 구현
- **결과**: 한글 타이핑 환경에 최적화된 정밀한 측정 엔진을 구축하여 서비스 신뢰도 확보
